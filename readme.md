# J-GAMES STUDIO — Underground Hub v3

Hub estático de proyectos con seis juegos completos, perfil local, economía JCOIN compartida, puntuación, historial y soporte mediante Gmail.

## Estructura pública
- `/index.html` — hub de proyectos.
- `/projects/<juego>/index.html` — páginas/juegos independientes.
- `/account/register.html` — registro local demo.
- `/account/profile.html` — tarjeta ID, JCOINS, puntuación e historial.
- `/assets/` — estilos, scripts, datos, imágenes y banners.

## Estructura interna
- `/internal/admin/` — ajuste del perfil local; no sirve para usuarios remotos.
- `/docs/` — documentación de producción y despliegue.
